# Holy Angel University Coding Activities

Welcome to **Holy Angel University Coding Activities** repository! This repository is the compilation of all the coding activities we have done in this university for academic purposes. 

### Repository Structure

To better navigate the repository, the directories have the following hierarchy: **year-level -> course -> due-date** where due date is an activity. To better explain it, here is the structure in text format for visualization using the first year directory as an example.

```
root
| -- first-year
|    | -- course-1
|    |    | -- due-date-1
|    |    | -- due-date-2
|    |    | -- due-date-3
|    |    | -- ...
|    | -- course-2
|    |    | -- due-date-1
|    |    | -- due-date-2
|    |    | -- due-date-3
|    |    | -- ...
```

> [!NOTE]
> If two or more activities have the same due date, numbered subdirectories inside the due date directory will be created based on the order of their deadline.

### Documentation

Most directories have a **README.md** file which will show details regarding said directory.  The documentation in the markdown file could either be the description of the course or the instructions for an activity. **If an activity has no documentation, it is most likely because the instructions were not announced online**. All information is copy pasted from [HAU Instructure](https://hau.instructure.com/) for convenience.

### Solutions

Every activity has a file for my personal code solution on each problem. **There are mostly no comments in the source code files**, but I try to make it as readable as possible even without it. From time to time, I will go back to previous activities to improve the solutions as my programming abilities improve.
